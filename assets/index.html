What makes it different

Every design decision exists to help you think more clearly.

◈

Living Connections

Click an edge and see exactly why two nodes are connected — which ideas reference them together, how recently, how strongly. Not just that they're linked, but why.

⬡

One global graph

No folders. No vaults. No hierarchy. Everything you write lives in a single graph. Structure emerges from connections, not from where you decide to file things.

◉

Physics-based layout

Nodes repel, spring toward their connections, and settle into organic arrangements. Drag one and the whole network shifts. It feels like thinking, not filing.

◎

Temporal salience

Recently edited nodes glow. Old nodes fade. The graph is a live picture of what's active in your mind right now, not an equal-weight archive of everything you've ever written.

▣

Weight from attention

Node size grows with incoming references. The concepts your thinking keeps returning to become visually prominent automatically — a PageRank derived from your own mind.

◌

Native. Instant. Private.

No Electron. No cloud. No account. A native app with a bundled runtime that opens instantly. Your data stays on your machine in a local SQLite file. Always.

The philosophy

Folders fight how memory actually works.

Human memory is associative, not hierarchical. You access a memory by traversing related concepts — not by opening a mental drawer labeled "Physics > 20th Century." Folders impose a structure your brain never agreed to.


Bibbo bets that forcing you to answer "what does this connect to?" instead of "where does this go?" produces a more honest map of how you actually understand things. The constraint isn't a limitation. It's the point.

NOT BUILT

Plugins, themes, canvas, kanban, AI writing assistant, browser clipper, collaboration, cloud sync. We said no to all of it on purpose.

BUILT

Fast graph navigation. Physics-based layout. Local-first storage. Writing mode. Living Connections. One experience, not a platform.

THE BET

A tool that does one thing so well you never need to configure it is more valuable than a tool that can do everything if you set it up right.
